Taller – Ecuaciones Diferenciales de Primer Orden

Resolución paso a paso · Método de ecuaciones separables y exactas

Lineamientos generales

  • Taller realizado en grupos de tres estudiantes.
  • Para cada ejercicio se indica la regla aplicada y el modelo matemático con sus variables.
  • Se desarrolla el procedimiento paso a paso sin omitir procesos algebraicos.
  • El resultado final de cada ejercicio se presenta dentro de un recuadro rojo.

Ejercicio a

\[ (x^2 + 4x)\,dx + x\,e^{y}\,dy = 0 \]

Regla aplicada

Se utiliza el método de ecuaciones de variables separables. Una ecuación diferencial ordinaria de primer orden es separable cuando puede escribirse en la forma \(f(x)\,dx = g(y)\,dy\), permitiendo integrar ambos lados de forma independiente.

Modelo matemático

Sean \(x\) e \(y\) variables reales independientes. La ecuación diferencial:

\[ M(x,y)\,dx + N(x,y)\,dy = 0 \]

donde

  • \(M(x,y) = x^2 + 4x\) — función que multiplica a \(dx\).
  • \(N(x,y) = x\,e^{y}\) — función que multiplica a \(dy\).
  • \(C\) — constante de integración arbitraria.

Procedimiento

  1. Factorizar \(x\) en \(M\): \[ x(x + 4)\,dx + x\,e^{y}\,dy = 0 \]
  2. Dividir entre \(x\) (con \(x \neq 0\)): \[ (x + 4)\,dx + e^{y}\,dy = 0 \]
  3. Separar variables: \[ (x + 4)\,dx = -e^{y}\,dy \]
  4. Integrar ambos miembros: \[ \int (x + 4)\,dx = -\int e^{y}\,dy \]
  5. Resolver las integrales: \[ \frac{x^2}{2} + 4x = -e^{y} + C \]
  6. Reordenar sumando \(e^y\) a ambos lados: \[ \frac{x^2}{2} + 4x + e^{y} = C \]
Solución general: \[ \frac{x^2}{2} + 4x + e^{y} = C \]

Ejercicio b

\[ (x^2 + 8x)\,dx = -(x\,e^{y}\,dy) \]

Regla aplicada

Se utiliza el método de ecuaciones de variables separables. Al reorganizar la ecuación se obtiene una forma separable que permite integrar cada variable de forma independiente.

Modelo matemático

Sean \(x\) e \(y\) variables reales independientes. La ecuación diferencial:

\[ M(x,y)\,dx + N(x,y)\,dy = 0 \]

donde

  • \(M(x,y) = x^2 + 8x\) — función que multiplica a \(dx\).
  • \(N(x,y) = x\,e^{y}\) — función que multiplica a \(dy\).
  • \(C\) — constante de integración arbitraria.

Procedimiento

  1. Pasar el término derecho al lado izquierdo: \[ (x^2 + 8x)\,dx + x\,e^{y}\,dy = 0 \]
  2. Factorizar \(x\): \[ x(x + 8)\,dx + x\,e^{y}\,dy = 0 \]
  3. Dividir entre \(x\) (con \(x \neq 0\)): \[ (x + 8)\,dx + e^{y}\,dy = 0 \]
  4. Separar variables: \[ (x + 8)\,dx = -e^{y}\,dy \]
  5. Integrar ambos miembros: \[ \int (x + 8)\,dx = -\int e^{y}\,dy \]
  6. Resolver las integrales: \[ \frac{x^2}{2} + 8x = -e^{y} + C \]
  7. Reordenar: \[ \frac{x^2}{2} + 8x + e^{y} = C \]
Solución general: \[ \frac{x^2}{2} + 8x + e^{y} = C \]

Ejercicio c

\[ (x^2 + 4y^2)\,dx + y\,x\,dy = 0 \]

Regla aplicada

Se utiliza el método de ecuaciones diferenciales exactas con factor integrante. Cuando \(\partial M/\partial y \neq \partial N/\partial x\) pero la expresión \(\dfrac{\partial M/\partial y - \partial N/\partial x}{N}\) depende únicamente de \(x\), se puede encontrar un factor integrante \(\mu(x) = e^{\int h(x)\,dx}\) que convierte la ecuación en exacta.

Modelo matemático

Sean \(x\) e \(y\) variables reales independientes. La ecuación diferencial:

\[ M(x,y)\,dx + N(x,y)\,dy = 0 \]

donde

  • \(M(x,y) = x^2 + 4y^2\) — función que multiplica a \(dx\).
  • \(N(x,y) = xy\) — función que multiplica a \(dy\).
  • \(\mu(x)\) — factor integrante dependiente sólo de \(x\).
  • \(F(x,y)\) — función potencial tal que \(dF = 0\).
  • \(C\) — constante de integración arbitraria.

Procedimiento

  1. Verificar si la ecuación es exacta: \[ \frac{\partial M}{\partial y} = 8y, \qquad \frac{\partial N}{\partial x} = y \] Como \(8y \neq y\), la ecuación no es exacta.
  2. Calcular la función auxiliar \(h(x)\): \[ h(x) = \frac{\dfrac{\partial M}{\partial y} - \dfrac{\partial N}{\partial x}}{N} = \frac{8y - y}{xy} = \frac{7y}{xy} = \frac{7}{x} \] Depende sólo de \(x\) → existe factor integrante \(\mu(x)\).
  3. Calcular el factor integrante: \[ \mu(x) = e^{\int \frac{7}{x}\,dx} = e^{7\ln|x|} = x^7 \]
  4. Multiplicar toda la ecuación por \(\mu = x^7\): \[ (x^9 + 4x^7 y^2)\,dx + x^8 y\,dy = 0 \]
  5. Verificar exactitud con la ecuación transformada: \[ \frac{\partial}{\partial y}(x^9 + 4x^7 y^2) = 8x^7 y, \qquad \frac{\partial}{\partial x}(x^8 y) = 8x^7 y \quad \checkmark \]
  6. Encontrar la función potencial \(F(x,y)\) integrando \(M^*\) respecto a \(x\): \[ F = \int (x^9 + 4x^7 y^2)\,dx = \frac{x^{10}}{10} + \frac{x^8 y^2}{2} + g(y) \]
  7. Determinar \(g(y)\) derivando respecto a \(y\) e igualando a \(N^*\): \[ \frac{\partial F}{\partial y} = x^8 y + g'(y) = x^8 y \implies g'(y) = 0 \implies g(y) = K \]
  8. Escribir la solución implícita \(F(x,y) = C\): \[ \frac{x^{10}}{10} + \frac{x^8 y^2}{2} = C \] Multiplicando por 10: \[ x^{10} + 5x^8 y^2 = C \]
Solución general: \[ x^{10} + 5x^8 y^2 = C \]

Ejercicio d

\[ y\,x\,dy = -\bigl((x^2 + 6y^2)\,dx\bigr) \]

Regla aplicada

Se utiliza el método de ecuaciones diferenciales exactas con factor integrante. Al igual que en el ejercicio anterior, se comprueba que la ecuación no es exacta, se calcula el factor integrante \(\mu(x)\) y se resuelve la ecuación transformada.

Modelo matemático

Sean \(x\) e \(y\) variables reales independientes. La ecuación diferencial:

\[ M(x,y)\,dx + N(x,y)\,dy = 0 \]

donde

  • \(M(x,y) = x^2 + 6y^2\) — función que multiplica a \(dx\).
  • \(N(x,y) = xy\) — función que multiplica a \(dy\).
  • \(\mu(x)\) — factor integrante dependiente sólo de \(x\).
  • \(F(x,y)\) — función potencial tal que \(dF = 0\).
  • \(C\) — constante de integración arbitraria.

Procedimiento

  1. Reorganizar la ecuación en forma estándar: \[ (x^2 + 6y^2)\,dx + xy\,dy = 0 \]
  2. Verificar si la ecuación es exacta: \[ \frac{\partial M}{\partial y} = 12y, \qquad \frac{\partial N}{\partial x} = y \] Como \(12y \neq y\), la ecuación no es exacta.
  3. Calcular la función auxiliar \(h(x)\): \[ h(x) = \frac{\dfrac{\partial M}{\partial y} - \dfrac{\partial N}{\partial x}}{N} = \frac{12y - y}{xy} = \frac{11y}{xy} = \frac{11}{x} \] Depende sólo de \(x\) → existe factor integrante \(\mu(x)\).
  4. Calcular el factor integrante: \[ \mu(x) = e^{\int \frac{11}{x}\,dx} = e^{11\ln|x|} = x^{11} \]
  5. Multiplicar toda la ecuación por \(\mu = x^{11}\): \[ (x^{13} + 6x^{11} y^2)\,dx + x^{12} y\,dy = 0 \]
  6. Verificar exactitud con la ecuación transformada: \[ \frac{\partial}{\partial y}(x^{13} + 6x^{11} y^2) = 12x^{11} y, \qquad \frac{\partial}{\partial x}(x^{12} y) = 12x^{11} y \quad \checkmark \]
  7. Encontrar la función potencial \(F(x,y)\): \[ F = \int (x^{13} + 6x^{11} y^2)\,dx = \frac{x^{14}}{14} + \frac{x^{12} y^2}{2} + g(y) \]
  8. Determinar \(g(y)\): \[ \frac{\partial F}{\partial y} = x^{12} y + g'(y) = x^{12} y \implies g'(y) = 0 \implies g(y) = K \]
  9. Escribir la solución implícita \(F(x,y) = C\): \[ \frac{x^{14}}{14} + \frac{x^{12} y^2}{2} = C \] Multiplicando por 14: \[ x^{14} + 7x^{12} y^2 = C \]
Solución general: \[ x^{14} + 7x^{12} y^2 = C \]

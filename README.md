# ejericio
📋 Descripción de la actividad El estudiante deberá crear una aplicación en Angular que: • Consuma la API pública de Rick and Morty. • Muestre una lista de personajes. • Visualice como mínimo:   – Imagen   – Nombre   – Estado (Alive/Dead/Unknown)   – Especie • Organice los personajes en tarjetas (cards). • Suba el proyecto a GitHub.

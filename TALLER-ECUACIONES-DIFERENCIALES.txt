ECUACIГ“N DIFERENCIAL EXACTA
----------------------------
Una ecuaciГіn M(x,y) dx + N(x,y) dy = 0 es EXACTA si y solo si:

        вҲӮM/вҲӮy = вҲӮN/вҲӮx

En ese caso existe F(x,y) tal que:
  dF = M dx + N dy  вҶ’  F(x,y) = C  (soluciГіn implГӯcita)

Para encontrar F(x,y):
  1. Integrar M respecto a x:  F = вҲ« M dx + g(y)
  2. Derivar F respecto a y e igualar a N para hallar g'(y)
  3. Integrar g'(y) para obtener g(y)

FACTOR INTEGRANTE Ој(x)
------------------------
Si la ecuaciГіn NO es exacta, se busca un factor integrante Ој
que depende sГіlo de x, usando la fГіrmula:

        f(x) = ( вҲӮM/вҲӮy вҲ’ вҲӮN/вҲӮx ) / N

Si f(x) depende SГ“LO de x, entonces:

        Ој(x) = e^( вҲ« f(x) dx )

Al multiplicar la ecuaciГіn original por Ој(x), la nueva
ecuaciГіn MвӮҒ dx + NвӮҒ dy = 0 SГҚ es exacta y se resuelve
con el mГ©todo de ecuaciones exactas.

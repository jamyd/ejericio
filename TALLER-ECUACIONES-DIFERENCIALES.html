Instrucciones Generales

Para cada ejercicio el estudiante deberá:

  1. Explicar la regla aplicada, indicando claramente cuál se utilizó para el desarrollo del ejercicio y justificando su uso.
  2. Definir el modelo matemático, explicando cada una de sus variables.
    Ejemplo: Sean a y b dos números reales. Al aplicar la operación de adición sobre estos valores se obtiene un número real c, tal que: a + b = c.
  3. Desarrollar el procedimiento paso a paso, incluyendo la explicación por escrito correspondiente en cada etapa, sin omitir procesos algebraicos, polinómicos, de integración, trigonométricos o similares.
  4. Presentar el resultado final dentro de un recuadro de color rojo.
Marco Teórico

Ecuación diferencial exacta. Una ecuación de la forma M(x, y) dx + N(x, y) dy = 0 es exacta si y solo si ∂M/∂y = ∂N/∂x. En ese caso, existe una función F(x, y) tal que dF = M dx + N dy, y la solución general es F(x, y) = C.

Factor integrante μ(x). Cuando la ecuación no es exacta, se busca un factor integrante μ(x) que depende sólo de x. Se calcula la expresión auxiliar:

f(x) = ( ∂M/∂y − ∂N/∂x ) ÷ N

Si f(x) depende únicamente de x, entonces:

μ(x) = e∫ f(x) dx

Al multiplicar la ecuación original por μ(x), la nueva ecuación M1 dx + N1 dy = 0 sí es exacta y se resuelve de forma directa.

Ejercicios
a)  (x + 4x) dx + x ey dy = 0  →  5x dx + x ey dy = 0
a) Regla aplicada — Factor integrante μ(x)

Se aplica el método del factor integrante dependiente de x porque, al evaluar la condición de exactitud, se obtiene ∂M/∂y ≠ ∂N/∂x. La expresión auxiliar f(x) = (∂M/∂y − ∂N/∂x) / N resulta ser una función exclusivamente de la variable x, lo que garantiza la existencia de un factor integrante μ(x). Al multiplicar la ecuación por μ(x) se obtiene una ecuación exacta que puede resolverse integrando directamente.

b) Modelo matemático

Sea la ecuación diferencial de primer orden escrita en la forma estándar M(x, y) dx + N(x, y) dy = 0, donde:

M(x, y) = 5x   → coeficiente que multiplica al diferencial dx; es una función lineal de x.

N(x, y) = x ey   → coeficiente que multiplica al diferencial dy; combina la variable x con la función exponencial de y.

La variable independiente es x y la variable dependiente (función implícita de x) es y. La constante C representa la familia de curvas solución.

c) Procedimiento paso a paso
  1. Verificar si la ecuación es exacta.
    Se calculan las derivadas parciales cruzadas:
    ∂M/∂y = ∂(5x)/∂y = 0 ∂N/∂x = ∂(x ey)/∂x = ey Como 0 ≠ ey, la ecuación NO es exacta en su forma original.
  2. Calcular la expresión auxiliar f(x).
    Se forma el cociente:
    f(x) = ( ∂M/∂y − ∂N/∂x ) / N = ( 0 − ey ) / ( x ey ) = −ey / (x ey) = −1/x El resultado depende únicamente de x. ✓ Existe μ(x).
  3. Calcular el factor integrante μ(x).
    μ(x) = e∫ f(x) dx = e∫ (−1/x) dx = e−ln|x| = eln|x|−1 = 1/x
  4. Multiplicar la ecuación original por μ = 1/x.
    (1/x) • [ 5x dx + x ey dy = 0 ] 5 dx + ey dy = 0
  5. Verificar exactitud de la nueva ecuación.
    M1 = 5  →  ∂M1/∂y = 0 N1 = ey  →  ∂N1/∂x = 0 Como ∂M1/∂y = ∂N1/∂x = 0, la ecuación ES EXACTA. ✓
  6. Integrar M1 respecto a x para obtener F(x, y).
    F = ∫ M1 dx = ∫ 5 dx = 5x + g(y) donde g(y) es una función arbitraria de y que se determina a continuación.
  7. Determinar g(y) usando ∂F/∂y = N1.
    ∂F/∂y = g'(y) = ey  →  g(y) = ∫ ey dy = ey + C1 La constante C1 se absorbe en la constante general C.
  8. Escribir la solución general.
    F(x, y) = 5x + ey = C
Resultado final
5x + ey = C
b)  (x + 8x) dx = −(x ey dy)  →  9x dx + x ey dy = 0
a) Regla aplicada — Factor integrante μ(x)

Al reescribir la ecuación en la forma estándar y verificar la condición de exactitud, se obtiene ∂M/∂y = 0 y ∂N/∂x = ey, confirmando que no es exacta. La expresión auxiliar f(x) = −1/x depende sólo de x, por lo que se aplica el método del factor integrante μ(x). El factor μ = 1/x transforma la ecuación en una exacta resoluble por integración directa.

b) Modelo matemático

La ecuación queda en la forma M dx + N dy = 0 con:

M(x, y) = 9x  → función lineal de x; coeficiente del diferencial dx.

N(x, y) = x ey  → producto de x y la función exponencial ey; coeficiente del diferencial dy.

x: variable independiente.   y: variable dependiente.   C: constante de integración que representa la familia de soluciones.

c) Procedimiento paso a paso
  1. Verificar exactitud.
    ∂M/∂y = ∂(9x)/∂y = 0 ∂N/∂x = ∂(x ey)/∂x = ey 0 ≠ ey → NO es exacta.
  2. Expresión auxiliar f(x).
    f(x) = ( 0 − ey ) / ( x ey ) = −1/x Depende sólo de x. ✓
  3. Factor integrante.
    μ(x) = e∫(−1/x)dx = e−ln|x| = 1/x
  4. Multiplicar por μ = 1/x.
    (1/x) • [ 9x dx + x ey dy ] = 0  →  9 dx + ey dy = 0
  5. Verificar exactitud.
    ∂(9)/∂y = 0  =  ∂(ey)/∂x = 0 ES EXACTA. ✓
  6. Integrar M1 = 9 respecto a x.
    F = ∫ 9 dx = 9x + g(y)
  7. Determinar g(y).
    ∂F/∂y = g'(y) = ey  →  g(y) = ey
  8. Solución general.
    F(x, y) = 9x + ey = C
Resultado final
9x + ey = C
c)  (x + 4y2) dx + yx dy = 0
a) Regla aplicada — Factor integrante μ(x)

Al verificar la condición de exactitud se obtiene ∂M/∂y = 8y y ∂N/∂x = y, valores distintos, por lo que la ecuación no es exacta. Se calcula la expresión:

f(x) = ( ∂M/∂y − ∂N/∂x ) / N = (8y − y) / (xy) = 7/x

Como f(x) = 7/x depende sólo de x, se aplica el método del factor integrante μ(x) = x7. La ecuación resultante es exacta y se resuelve integrando M1 respecto a x y determinando la función g(y).

b) Modelo matemático

La ecuación está en la forma M dx + N dy = 0 donde:

M(x, y) = x + 4y2  → suma de la variable independiente x y cuatro veces el cuadrado de la variable dependiente y.

N(x, y) = xy  → producto de ambas variables; representa la interacción entre x e y en el término dy.

x: variable independiente.   y: variable dependiente.   C: constante arbitraria (familia de curvas solución).

c) Procedimiento paso a paso
  1. Verificar exactitud.
    ∂M/∂y = ∂(x + 4y2)/∂y = 8y ∂N/∂x = ∂(xy)/∂x = y 8y ≠ y → NO es exacta.
  2. Expresión auxiliar f(x).
    f(x) = ( 8y − y ) / (xy) = 7y / (xy) = 7/x Depende sólo de x. ✓
  3. Factor integrante.
    μ(x) = e∫(7/x)dx = e7 ln|x| = eln |x|7 = x7
  4. Multiplicar la ecuación por μ = x7.
    x7 • [ (x + 4y2)dx + xy dy ] = 0 ( x8 + 4x7y2 ) dx + x8y dy = 0
  5. Verificar exactitud de la nueva ecuación.
    M1 = x8 + 4x7y2  →  ∂M1/∂y = 8x7y N1 = x8y  →  ∂N1/∂x = 8x7y ∂M1/∂y = ∂N1/∂x = 8x7y → ES EXACTA. ✓
  6. Integrar M1 respecto a x.
    F = ∫ ( x8 + 4x7y2 ) dx = x9/9 + 4 • (x8/8) • y2 + g(y) = x9/9 + x8y2/2 + g(y)
  7. Determinar g(y) usando ∂F/∂y = N1.
    ∂F/∂y = x8y + g'(y) = x8y  →  g'(y) = 0  →  g(y) = C1 La función g(y) es constante, lo que confirma que F(x, y) es correcta.
  8. Solución general.
    F(x, y) = x9/9 + x8y2/2 = C Multiplicando por 18 para eliminar denominadores: 2x9 + 9x8y2 = K
Resultado final
x9/9 + x8y2/2 = C
(equivalente: 2x9 + 9x8y2 = K)
d)  yx dy = −((x + 6y2) dx)  →  (x + 6y2) dx + xy dy = 0
a) Regla aplicada — Factor integrante μ(x)

Al reescribir la ecuación en la forma estándar M dx + N dy = 0 y verificar la condición de exactitud se obtiene ∂M/∂y = 12y y ∂N/∂x = y, valores distintos: la ecuación no es exacta. La expresión auxiliar:

f(x) = (12y − y) / (xy) = 11/x

depende exclusivamente de x, por lo que se aplica el método del factor integrante μ(x) = x11. La nueva ecuación es exacta y se resuelve integrando directamente.

b) Modelo matemático

Forma estándar M dx + N dy = 0 con:

M(x, y) = x + 6y2  → suma de la variable independiente x y seis veces el cuadrado de y; coeficiente de dx.

N(x, y) = xy  → producto de las variables x e y; coeficiente de dy.

x: variable independiente.   y: variable dependiente (implícita).   C: constante de integración.

c) Procedimiento paso a paso
  1. Verificar exactitud.
    ∂M/∂y = ∂(x + 6y2)/∂y = 12y ∂N/∂x = ∂(xy)/∂x = y 12y ≠ y → NO es exacta.
  2. Expresión auxiliar f(x).
    f(x) = ( 12y − y ) / (xy) = 11y / (xy) = 11/x Depende sólo de x. ✓
  3. Factor integrante.
    μ(x) = e∫(11/x)dx = e11 ln|x| = eln|x|11 = x11
  4. Multiplicar la ecuación por μ = x11.
    x11 • [ (x + 6y2)dx + xy dy ] = 0 ( x12 + 6x11y2 ) dx + x12y dy = 0
  5. Verificar exactitud de la nueva ecuación.
    M1 = x12 + 6x11y2  →  ∂M1/∂y = 12x11y N1 = x12y  →  ∂N1/∂x = 12x11y ∂M1/∂y = ∂N1/∂x = 12x11y → ES EXACTA. ✓
  6. Integrar M1 respecto a x.
    F = ∫ ( x12 + 6x11y2 ) dx = x13/13 + 6 • (x12/12) • y2 + g(y) = x13/13 + x12y2/2 + g(y)
  7. Determinar g(y) usando ∂F/∂y = N1.
    ∂F/∂y = x12y + g'(y) = x12y  →  g'(y) = 0  →  g(y) = C1 g(y) es constante, lo que verifica que F(x, y) es la función potencial correcta.
  8. Solución general.
    F(x, y) = x13/13 + x12y2/2 = C Multiplicando por 26 para eliminar denominadores: 2x13 + 13x12y2 = K
Resultado final
x13/13 + x12y2/2 = C
(equivalente: 2x13 + 13x12y2 = K)

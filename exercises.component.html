Taller de Ecuaciones Diferenciales

Solución de ecuaciones diferenciales ordinarias de primer orden mediante separación de variables y factor integrante.

Ejercicio a)

(x + 4x) dx + x ey dy = 0

Regla aplicada: Ecuación Diferencial Separable

Se utiliza la regla de separación de variables porque, tras simplificar, es posible escribir la ecuación de modo que todos los términos con x queden en un miembro y todos los términos con y en el otro, para integrar cada lado de forma independiente.

Modelo matemático

Sean x e y variables reales independientes. La ecuación establece una relación diferencial de la forma:

M(x, y) dx + N(x, y) dy = 0

donde M(x, y) = x + 4x = 5x y N(x, y) = x ey.

Procedimiento paso a paso

  1. Simplificar M:

    (x + 4x) dx + x ey dy = 0

    5x dx + x ey dy = 0

  2. Dividir ambos miembros entre x (x ≠ 0):

    5 dx + ey dy = 0

  3. Separar variables:

    ey dy = −5 dx

  4. Integrar ambos miembros:

    ∫ ey dy = ∫ −5 dx

    ey = −5x + C

Resultado:

ey + 5x = C

Ejercicio b)

(x + 8x) dx = −(x ey dy)

Regla aplicada: Ecuación Diferencial Separable

Se aplica separación de variables porque, al simplificar los coeficientes y dividir entre x, los términos con x y con y pueden aislarse en lados opuestos de la ecuación para ser integrados por separado.

Modelo matemático

Sean x e y variables reales. Reescribiendo la ecuación en la forma estándar:

M(x, y) dx + N(x, y) dy = 0

donde M(x, y) = x + 8x = 9x y N(x, y) = x ey.

Procedimiento paso a paso

  1. Reescribir en forma estándar:

    (x + 8x) dx + x ey dy = 0

    9x dx + x ey dy = 0

  2. Dividir entre x (x ≠ 0):

    9 dx + ey dy = 0

  3. Separar variables:

    ey dy = −9 dx

  4. Integrar ambos miembros:

    ∫ ey dy = ∫ −9 dx

    ey = −9x + C

Resultado:

ey + 9x = C

Ejercicio c)

(x + 4y2) dx + yx dy = 0

Regla aplicada: Ecuación Exacta con Factor Integrante μ(x)

La ecuación no es exacta en su forma original, ya que ∂M/∂y ≠ ∂N/∂x. Sin embargo, el cociente (∂M/∂y − ∂N/∂x) / N depende únicamente de x, lo que permite hallar un factor integrante μ(x) = xn para convertirla en una ecuación exacta e integrable.

Modelo matemático

Sean x e y variables reales con x ≠ 0. Se busca una función F(x, y) tal que:

dF = ∂F/∂x dx + ∂F/∂y dy = 0

donde M(x, y) = x + 4y2, N(x, y) = yx, y la solución general satisface F(x, y) = C.

Procedimiento paso a paso

  1. Verificar exactitud:

    ∂M/∂y = 8y

    ∂N/∂x = y

    Como 8y ≠ y, la ecuación no es exacta.

  2. Calcular el factor integrante:

    (∂M/∂y − ∂N/∂x) / N

    = (8y − y) / (yx)

    = 7y / (yx)

    = 7/x

    El resultado depende sólo de x, por lo que el factor integrante es:

    μ(x) = e∫(7/x)dx = e7 ln|x| = x7

  3. Multiplicar la ecuación por μ = x7:

    (x8 + 4x7y2) dx + x8y dy = 0

    Verificar exactitud: ∂M*/∂y = 8x7y = ∂N*/∂x = 8x7y ✓

  4. Hallar F integrando M* respecto a x:

    F = ∫(x8 + 4x7y2) dx = x9/9 + x8y2/2 + g(y)

  5. Determinar g(y) derivando F respecto a y e igualando a N*:

    ∂F/∂y = x8y + g'(y) = x8y

    g'(y) = 0 → g(y) = C1

Resultado:

x9/9 + x8y2/2 = C

Ejercicio d)

yx dy = −((x + 6y2) dx)

Regla aplicada: Ecuación Exacta con Factor Integrante μ(x)

Al igual que el ejercicio anterior, la ecuación en su forma estándar no es exacta (∂M/∂y ≠ ∂N/∂x). El cociente (∂M/∂y − ∂N/∂x) / N depende exclusivamente de x, lo que permite obtener un factor integrante μ(x) = xn y transformar la ecuación en una ecuación exacta resoluble.

Modelo matemático

Sean x e y variables reales con x ≠ 0. Se busca F tal que:

dF = ∂F/∂x dx + ∂F/∂y dy = 0

donde M(x, y) = x + 6y2, N(x, y) = yx, y la solución general satisface F(x, y) = C.

Procedimiento paso a paso

  1. Reescribir en forma estándar:

    (x + 6y2) dx + yx dy = 0

  2. Verificar exactitud:

    ∂M/∂y = 12y

    ∂N/∂x = y

    Como 12y ≠ y, la ecuación no es exacta.

  3. Calcular el factor integrante:

    (∂M/∂y − ∂N/∂x) / N

    = (12y − y) / (yx)

    = 11y / (yx)

    = 11/x

    El resultado depende sólo de x, por lo que el factor integrante es:

    μ(x) = e∫(11/x)dx = e11 ln|x| = x11

  4. Multiplicar la ecuación por μ = x11:

    (x12 + 6x11y2) dx + x12y dy = 0

    Verificar exactitud: ∂M*/∂y = 12x11y = ∂N*/∂x = 12x11y ✓

  5. Hallar F integrando M* respecto a x:

    F = ∫(x12 + 6x11y2) dx = x13/13 + x12y2/2 + g(y)

  6. Determinar g(y) derivando F respecto a y e igualando a N*:

    ∂F/∂y = x12y + g'(y) = x12y

    g'(y) = 0 → g(y) = C1

Resultado:

x13/13 + x12y2/2 = C

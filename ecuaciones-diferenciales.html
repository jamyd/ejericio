Taller de Ecuaciones Diferenciales

Solución paso a paso  ·  Ecuaciones Exactas y Separables

Ejercicio a)
a
(x + 4x) dx + x ey dy = 0
Regla aplicada

Ecuación diferencial separable. Una ecuación es separable cuando puede escribirse en la forma M(x) dx + N(y) dy = 0, permitiendo integrar cada lado independientemente:

∫M(x) dx + ∫N(y) dy = C
Modelo matemático

Sean x la variable independiente e y la variable dependiente, ambas funciones reales y diferenciables. La ecuación expresa la relación diferencial:

(x + 4x) dx + x ey dy = 0
Procedimiento paso a paso
  1. Simplificar el primer factor
    x + 4x = 5x

    La ecuación queda:

    5x dx + x ey dy = 0
  2. Dividir toda la ecuación por x (con x ≠ 0)
    5x dxx + x ey dyx = 0
    ⟹
    5 dx + ey dy = 0
  3. Separar variables
    5 dx = −ey dy
  4. Integrar ambos lados
    ∫5 dx = −∫ey dy
    ⟹
    5x = −ey + C
  5. Reordenar
    5x + ey = C
Resultado final
5x + ey = C
Ejercicio b)
b
(x + 8x) dx = −(x ey dy)
Regla aplicada

Ecuación diferencial separable. Se lleva la ecuación a la forma estándar M dx + N dy = 0 y se integra cada lado de forma independiente.

Modelo matemático

La ecuación dada es equivalente a la forma estándar con M(x) y N(y) separados:

(x + 8x) dx + x ey dy = 0
Procedimiento paso a paso
  1. Llevar al lado izquierdo el término con dy
    (x + 8x) dx + x ey dy = 0
  2. Simplificar el primer factor
    x + 8x = 9x

    La ecuación queda:

    9x dx + x ey dy = 0
  3. Dividir por x (con x ≠ 0)
    9 dx + ey dy = 0
  4. Separar variables
    9 dx = −ey dy
  5. Integrar ambos lados
    ∫9 dx = −∫ey dy
    ⟹
    9x = −ey + C
  6. Reordenar
    9x + ey = C
Resultado final
9x + ey = C
Ejercicio c)
c
(x + 4y2) dx + yx dy = 0
Regla aplicada

Factor integrante dependiente solo de x. Cuando la ecuación M dx + N dy = 0 no es exacta, pero la expresión

f(x) = ∂M/∂y − ∂N/∂x N

depende únicamente de x, el factor integrante μ(x) = e∫f(x)dx convierte la ecuación en exacta.

Modelo matemático

Sean M(x,y) = x + 4y2 y N(x,y) = xy las funciones que definen la ecuación:

M dx + N dy = 0

Una solución implícita F(x,y) = C existe si ∂F/∂x = μM y ∂F/∂y = μN.

Procedimiento paso a paso
  1. Identificar M y N
    M = x + 4y2
    N = xy
  2. Verificar exactitud
    ∂M/∂y = 8y
    ∂N/∂x = y

    Como 8y ≠ y, la ecuación no es exacta.

  3. Calcular el factor integrante
    f(x) = 8y − y xy = 7y xy = 7 x

    Como f(x) depende solo de x:

    μ(x) = e∫(7/x)dx = e7 ln|x| = x7
  4. Multiplicar la ecuación por μ = x7
    (x8 + 4x7y2) dx + x8y dy = 0
  5. Verificar exactitud de la nueva ecuación
    ∂M̃/∂y = 8x7y
    ∂Ñ/∂x = 8x7y
    ✓ Exacta
  6. Hallar la función potencial F(x,y)
    F = ∫(x8 + 4x7y2) dx = x9 9 + 4x8y2 8 + g(y) = x9 9 + x8y2 2 + g(y)
  7. Determinar g(y)
    ∂F/∂y = x8y + g′(y) = Ñ = x8y
    ⟹
    g′(y) = 0  ⟹  g(y) = k
  8. Escribir la solución general
    x9 9 + x8y2 2 = C

    Multiplicando por 18 para eliminar fracciones:

    2x9 + 9x8y2 = C
Resultado final
2x9 + 9x8y2 = C
Ejercicio d)
d
yx dy = −((x + 6y2) dx)
Regla aplicada

Factor integrante dependiente solo de x. Se aplica el mismo método del ejercicio c): si la ecuación no es exacta pero la expresión (∂M/∂y − ∂N/∂x) / N depende solo de x, el factor integrante μ(x) transforma la ecuación en exacta.

Modelo matemático

Reescribiendo en forma estándar M dx + N dy = 0:

(x + 6y2) dx + xy dy = 0

con M(x,y) = x + 6y2 y N(x,y) = xy.

Procedimiento paso a paso
  1. Reescribir en forma estándar
    yx dy + (x + 6y2) dx = 0
    ⟹
    (x + 6y2) dx + xy dy = 0
  2. Identificar M y N
    M = x + 6y2
    N = xy
  3. Verificar exactitud
    ∂M/∂y = 12y
    ∂N/∂x = y

    Como 12y ≠ y, la ecuación no es exacta.

  4. Calcular el factor integrante
    f(x) = 12y − y xy = 11y xy = 11 x

    Como f(x) depende solo de x:

    μ(x) = e∫(11/x)dx = e11 ln|x| = x11
  5. Multiplicar la ecuación por μ = x11
    (x12 + 6x11y2) dx + x12y dy = 0
  6. Verificar exactitud de la nueva ecuación
    ∂M̃/∂y = 12x11y
    ∂Ñ/∂x = 12x11y
    ✓ Exacta
  7. Hallar la función potencial F(x,y)
    F = ∫(x12 + 6x11y2) dx = x13 13 + 6x12y2 12 + g(y) = x13 13 + x12y2 2 + g(y)
  8. Determinar g(y)
    ∂F/∂y = x12y + g′(y) = Ñ = x12y
    ⟹
    g′(y) = 0  ⟹  g(y) = k
  9. Escribir la solución general
    x13 13 + x12y2 2 = C

    Multiplicando por 26 para eliminar fracciones:

    2x13 + 13x12y2 = C
Resultado final
2x13 + 13x12y2 = C
